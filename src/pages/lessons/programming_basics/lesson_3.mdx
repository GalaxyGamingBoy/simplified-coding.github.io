# Δομές Ελέγχου

## Συνθήκες

Οι "συνθήκες" (conditional statements) είναι τμήματα κώδικα προγράμματος που επιτρέπουν στον υπολογιστή να πάρει αποφάσεις βάσει κάποιων συνθηκών.
Απλά, ανάλογα με το αν μια συγκεκριμένη συνθήκη είναι αληθής ή ψευδής, ο υπολογιστής εκτελεί διαφορετικό κώδικα. Ας δούμε τις τρεις βασικές μορφές συνθηκών:

### Δομή if (if statement)

Σε αυτήν τη δομή,
εκτελείται ένα τμήμα κώδικα αν η συνθήκη είναι αληθής.

ηλικία = 20
εάν ηλικία είναι > ή = 18
  τότε γράψε "Είσαι ενήλικας!"

### Δομή if-else if (else if statement)

Σε αυτήν τη δομή,
ελέγχεται μια σειρά από συνθήκες,
και αν μία από αυτές είναι αληθής,
εκτελείται ο αντίστοιχος κώδικας.
Αν καμία από τις συνθήκες δεν είναι αληθής, τότε δεν εκτελείται κανένα τμήμα κώδικα.

score = 85
εάν score > ή = 90
  γράψε "Άριστα!"
αλλιώς εάν score > ή = 80
  γράψε "Πολύ καλά!"
αλλιώς εάν score > ή = 70
  γράψε "Καλά"

### Δομή if-else if-else (else statement)

Σε αυτήν τη δομή, ελέγχεται μια σειρά από συνθήκες,
και εκτελείται ο κώδικας της πρώτης αληθούς συνθήκης.
Αν καμία από τις συνθήκες δεν είναι αληθής, τότε εκτελείται ο κώδικας στο "else" μέρος.


grade = "B"
εάν grade = "A"
  γράψε "Άριστα!"
αλλιώς εάν grade = "B"
  γράψε "Πολύ καλά!"
αλλιώς
  γράψε "Καλά"

## Loops

Τα loops είναι δομές στον προγραμματισμό που μας επιτρέπουν να εκτελούμε τμήματα κώδικα επαναληπτικά μέχρι να ικανοποιηθεί μια συγκεκριμένη συνθήκη.
Υπάρχουν δύο βασικοί τύποι loop

### For Loop
Χρησιμοποιείται όταν γνωρίζετε τον αριθμό των επαναλήψεων που θέλετε να κάνετε. Για παράδειγμα:

για i στο range(5):
    γράψε "Επανάληψη" i

Αυτό το loop θα εκτελέσει τον κώδικα μέσα σε αυτόν 5 φορές, με το `i` να αυξάνεται από το 0 έως το 4.

### While Loop
Χρησιμοποιείται όταν δεν ξέρετε προκαταβολικά πόσες επαναλήψεις θα χρειαστούν,
αλλά γνωρίζετε μια συνθήκη που πρέπει να ικανοποιηθεί για να σταματήσετε το loop. Για παράδειγμα:

μέτρηση = 0
όσο μέτρηση μικρότερο από 5:
    γράψε "Επανάληψη" και μέτρηση
    μέτρηση + 1

Αυτό το loop θα εκτελέσει τον κώδικα μέσα σε αυτόν όσο η μεταβλητή `count` είναι μικρότερη από 5.

## Συναρτήσεις

Στον προγραμματισμό,
μια "συνάρτηση" είναι ένα αυτόνομο τμήμα κώδικα που εκτελεί μια συγκεκριμένη εργασία.
Οι συναρτήσεις είναι σαν μικρά προγράμματα εντός ενός μεγαλύτερου προγράμματος.
Σας επιτρέπουν να διασπάσετε τον κώδικά σας σε μικρότερα,
επαναχρησιμοποιήσιμα κομμάτια, πράγμα που καθιστά τον κώδικά σας πιο οργανωμένο και ευκολότερο στη συντήρηση.

Οι συναρτήσεις συνήθως έχουν τα ακόλουθα στοιχεία:

1. **Όνομα**: Ένα όνομα που αναγνωρίζει μοναδικά τη συνάρτηση. Χρησιμοποιείται για να καλέσετε τη συνάρτηση.

2. **Παράμετροι**: Τιμές που η συνάρτηση μπορεί να δεχτεί ως είσοδο. Αυτές είναι προαιρετικές, ανάλογα με τον σκοπό της συνάρτησης.

3. **Σώμα**: Ο πραγματικός κώδικας που καθορίζει τι κάνει η συνάρτηση. Αυτός ο κώδικας βρίσκεται εντός αγκυλών `{}` σε κάποιες γλώσσες όπως η JavaScript και η C++.

4. **Επιστρεφόμενη Τιμή**: Ορισμένες συναρτήσεις επιστρέφουν μια τιμή μετά την εκτέλεσή τους, η οποία μπορεί να χρησιμοποιηθεί από τον κώδικα που την καλεί.

Είναι σημαντικό να έχετε στον νου σας ότι οι συναρτήσεις σάς επιτρέπουν να καλείτε τον ίδιο κώδικα πολλές φορές,
κάνοντας τον κώδικά σας πιο αποδοτικό και ευκολότερο στη διαχείριση.
