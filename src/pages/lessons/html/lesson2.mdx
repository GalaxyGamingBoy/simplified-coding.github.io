# Μάθημα 2ο: Παράγραφοι, Τίτλοι, Links

## `p` element

Το p element είναι ένα στοιχείο που χρησιμοποιείται για να δημιουργήσει μια παράγραφο.

### Παράδειγμα

```
<p>Αυτό είναι μία παράγραφος.</p>
```

## `h` artibutes

Τα h attributes χρησιμοποιούνται για να δηλώσουμε το επίπεδο του τίτλου.

### Παράδειγμα

```
<h1>Αυτός είναι ένας τίτλος.</h1>
```
Τα h attributes χρησιμοποιούνται για να δηλώσουμε το επίπεδο του τίτλου.
Το h1 είναι μεγαλύτερος από το h2, το h2 είναι μεγαλύτερος από το h3 κλπ μέχρι το h6 που είναι ο μικρότερος τίτλος.

## `a` element

Το a element χρησιμοποιείται για να δημιουργήσει ένα link σε μια άλλη σελίδα.

### Παράδειγμα

```
<a href="https://simplifiedcoding.org">simplified Coding</a>
```

Βλέπουμε ότι το link έχει δύο μέρη. 
Το πρώτο μέρος είναι το `href` attribute. Το `href` attribute δηλώνει τη διεύθυνση της σελίδας στην οποία θα μας πάει το link. 
Το δεύτερο μέρος είναι το κείμενο του link. Αυτό είναι το κείμενο που θα εμφανίζεται στη σελίδα.
