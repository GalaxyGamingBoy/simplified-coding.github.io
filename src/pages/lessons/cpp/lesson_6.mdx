# Μάθημα 6: While loops

## While loops

Το while loop είναι ένα πολύ σημαντικό και χρήσιμο εργαλείο στη C++.
Μας επιτρέπει να επαναλαμβάνουμε ένα κομμάτι κώδικα όσο ισχύει μια συνθήκη.

Ας δούμε ένα παράδειγμα:

### Παράδειγμα

```cpp
#include <iostream>

using namespace std;

int main() {
    int i = 0;
    while (i <= 10) {
        cout << i << endl;
        i++;
    }
    return 0;
}
```

Αυτό το πρόγραμμα θα τυπώσει αριθμούς από το 0 μέχρι το 10. Ας αναλύσουμε το πρόγραμμα γραμμή προς γραμμή:

### `while (i <= 10)`

Αρχίζουμε την επανάληψη με την εντολή `while` και τη συνθήκη που θα ελέγχεται σε κάθε επανάληψη.
Η συνθήκη μας λέει πως όσο η μεταβλητή `i` είναι μικρότερη ή ίση του 10, θα εκτελείται ο κώδικας μέσα στο `while`.

### `cout << i << endl;`

Τυπώνουμε την τιμή της μεταβλητής `i`.

### `i++;`

Αυξάνουμε την τιμή της μεταβλητής `i` κατά 1.

## Do-while loops

Το do-while loop είναι παρόμοιο με το while loop, με τη διαφορά ότι ελέγχει τη συνθήκη στο τέλος της επανάληψης.
Αυτό σημαίνει πως θα εκτελεστεί ο κώδικας μια φορά ανεξάρτητος αν η συνθήκη είναι αληθής ή όχι και μετά θα
συνεχίσει αν η συνθήκη είναι αληθής.

Ας δούμε ένα παράδειγμα:

### Παράδειγμα

```cpp
#include <iostream>

using namespace std;

int main() {
    int i = 11;
    do {
        cout << i << endl;
        i++;
    } while (i <= 10);
    return 0;
}
```
