# Μάθημα 11: Συναρτήσεις

Οι συναρτήσεις είναι κομμάτια κώδικα που τρέχουν όταν τα καλούμε.
Μπορούμε και να τους δώσουμε τιμές (παραμέτρους) οι οποίες θα δώσουν ένα αποτέλεσμα.
Αυτό έχει ως αποτέλεσμα να μη χρειάζεται να γράφουμε τον ίδιο κώδικα πολλές φορές.

## Δημιουργία συνάρτησης

Η C++ μας δίνει τη συνάρτηση main() η οποία είναι τρέχει όταν αρχίσει το πρόγραμμα.
Μπορούμε να φτιάξουμε και τις δικές μας συναρτήσεις, έξω από αυτήν.

Για να φτιάξουμε μια συνάρτηση, γράφουμε `void`, το όνομα της και τις παραμέτρους της.

### Παραδείγματα

#### Χωρίς παραμέτρους

```cpp
#include <iostream>

using namespace std;

void printHello() {
    cout << "Γειά!" << endl;
}

int main() {
    // Κώδικας
}
```

#### Με παραμέτρους

```cpp
#include <iostream>

using namespace std;

void printHello(string name) {
    cout << "Γειά " << name << "!" << endl;
}

int main() {
    // Κώδικας
}
```

## Κλήση συνάρτησης

Για να καλέσουμε μια συνάρτηση, γράφουμε το όνομά της και τις παραμέτρους της.

### Παραδείγματα

#### Χωρίς παραμέτρους

```cpp
#include <iostream>

using namespace std;

void printHello() {
    cout << "Γειά!" << endl;
}

int main() {
    printHello();
}
```

#### Με παραμέτρους

```cpp
#include <iostream>

using namespace std;

void printHello(string name) {
    cout << "Γειά " << name << "!" << endl;
}

int main() {
    printHello("Γιώργο");
}
```
