# Μάθημα 4: Συνθήκες

Στη C++ μπορούμε να εκτελέσουμε κώδικα μόνο όταν μια συνθήκη αληθεύει.
Μια συνθήκη μπορεί να είναι για παράδειγμα μια σύγκριση μεταξύ δύο μεταβλητών.

Υπάρχουν 3 τύποι συνθηκών:

* Συνθήκη if
* Συνθήκη else
* Συνθήκη else if

## Συνθήκη if

Η συνθήκη if είναι η πιο απλή. Ελέγχει αν μια συνθήκη είναι αληθής.
Αν είναι, εκτελεί μόνο τότε κάποιες εντολές.

### Παράδειγμα

Στο παρακάτω παράδειγμα, θα εκτυπώσουμε ένα μήνυμα μόνο αν ο αριθμός είναι μεγαλύτερος από 5:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 6;
    if (x > 5) {
        cout << "Ο αριθμός είναι μεγαλύτερος από 5";
    }
    return 0;
}
```

Ας αναλύσουμε τα στοιχεία της συνθήκης:

#### Συνθήκη: `if (x > 5)`

Αυτό είναι το τμήμα της συνθήκης που ελέγχει αν ο αριθμός είναι μεγαλύτερος από 5.

#### Αγκύλες: `{ ... }`

Μέσα σε αυτές τις αγκύλες βάζουμε τις εντολές που θέλουμε να εκτελεστούν αν η συνθήκη είναι αληθής.
Οτιδήποτε βάλουμε εδώ θα εκτελεστεί μόνο αν η συνθήκη είναι αληθής
και οτιδήποτε βάλουμε εκτός αυτών των αγκυλών θα εκτελεστεί ανεξάρτητα από τη συνθήκη.

## Συνθήκη else

Η συνθήκη else τρέχει όταν η συνθήκη if είναι ψευδής.
Δεν είναι υποχρεωτική, αλλά μπορεί να χρησιμοποιηθεί για να εκτελέσει κώδικα
όταν η συνθήκη if είναι ψευδής αν το θέλουμε.

### Παράδειγμα

Στο παρακάτω παράδειγμα, θα εκτυπώσουμε ένα μήνυμα αν ο αριθμός είναι μεγαλύτερος από 5,
αλλά θα εκτυπώσουμε ένα διαφορετικό μήνυμα αν ο αριθμός είναι μικρότερος ή ίσος με 5:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 3;
    if (x > 5) {
        cout << "Ο αριθμός είναι μεγαλύτερος από 5";
    } else {
        cout << "Ο αριθμός είναι μικρότερος ή ίσος με 5";
    }
    return 0;
}
```

## Συνθήκη else if

Η συνθήκη else if βρίσκεται μεταξύ των συνθηκών if και else.
Χρησιμοποιείται για να ελέγξει μια νέα συνθήκη αν η προηγούμενη συνθήκη ήταν ψευδής.

Μπορούμε να έχουμε όσες συνθήκες else if θέλουμε.
Επιπλέον, πρέπει πάντα να είναι μια συνθήκη else if ακριβώς μετά μια συνθήκη if ή else if και
πριν από μια συνθήκη else.

### Παράδειγμα

Στο παρακάτω παράδειγμα, θα εκτυπώσουμε ένα μήνυμα αν ο αριθμός είναι μεγαλύτερος από 5,
αλλά θα εκτυπώσουμε ένα διαφορετικό μήνυμα αν ο αριθμός είναι μικρότερος ή ίσος με 5,
αλλά μεγαλύτερος από 0:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = -3;
    if (x > 5) {
        cout << "Ο αριθμός είναι μεγαλύτερος από 5";
    } else if (x > 0) {
        cout << "Ο αριθμός είναι μικρότερος ή ίσος με 5, αλλά μεγαλύτερος από 0";
    } else {
        cout << "Ο αριθμός είναι μικρότερος ή ίσος με 0";
    }
    return 0;
}
```
