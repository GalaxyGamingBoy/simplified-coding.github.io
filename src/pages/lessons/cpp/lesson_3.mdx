# Μάθημα 3: Μεταβλητές και Τύποι Δεδομένων

## Τύποι Δεδομένων

Πριν δούμε τις μεταβλητές στη C++, θα πρέπει να δούμε τους τύπους δεδομένων που υπάρχουν.

Υπάρχουν 5 βασικοί τύποι δεδομένων στη C++:

| Τύπος Δεδομένων | Περιγραφή              |
|-----------------|------------------------|
| integer         | Ακέραιος αριθμός       |
| double          | Δεκαδικός αριθμός      |
| character       | Ένας χαρακτήρας        |
| string          | Κείμενο                |
| boolean         | Τιμή αληθούς ή ψεύδους |

## Μεταβλητές

Οι μεταβλητές είναι ένας απλός αποθηκευτικός χώρος στη μνήμη, ο οποίος έχει έναν από τους 5 τύπους δεδομένων που είδαμε
πριν. Οι μεταβλητές έχουν ένα όνομα, το οποίο πρέπει να είναι μοναδικό στο πρόγραμμα, και μπορούν να πάρουν μια τιμή
από τον τύπο δεδομένων που έχουν.

### Ορισμός μεταβλητής

Για να ορίσουμε μια μεταβλητή, πρέπει να γράψουμε τον τύπο δεδομένων, το όνομα της μεταβλητής και την τιμή που θέλουμε
να πάρει η μεταβλητή. Αν δε θέλουμε να δώσουμε τιμή στη μεταβλητή, τότε απλά δε γράφουμε την τιμή.

#### Παραδείγματα

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 5;
    double y = 3.14;
    char z = 'a';
    string w = "Hello World!";
    bool t = true;
    return 0;
}
```

### Λειτουργίες μεταβλητών

Στο προηγούμενο μάθημα είχαμε μάθει διάφορες εργασίες με μαθηματικά. Μπορούμε να εφαρμόσουμε ότι μάθαμε να κάνουμε
με το `cout` σε μεταβλητές.

#### Παραδείγματα

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    int x = 5;
    int y = 3;
    int z = x + y;
    int w;
    cout << z << endl;
    cout << x != y << endl;
    cout << x != y && x == y << endl;
    w = sqrt(x);
    cout << w << endl;
}
```

## Είσοδος Δεδομένων

Μέχρι τώρα, έχουμε δίνει τιμές στις μεταβλητές μας μέσα στο πρόγραμμα.
Μπορούμε όμως να δίνουμε τιμές στις μεταβλητές μας και από τον χρήστη.
Για να το κάνουμε αυτό, χρησιμοποιούμε την εντολή `cin`.

### Παράδειγμα

```cpp
#include <iostream>

using namespace std;

int main() {
    int x;
    cout << "Εισάγετε έναν αριθμό: ";
    cin >> x;
    cout << "Εισάγατε: " << x << endl;
    return 0;
}
```
