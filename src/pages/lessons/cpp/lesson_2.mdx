# Μάθημα 2: C++ Math

## Πράξεις

Υπάρχουν 5 βασικές πράξεις στη C++:

| Πράξη              | Σύμβολο  |
|--------------------|----------|
| Πρόσθεση           | `+`      |
| Αφαίρεση           | `-`      |
| Πολλαπλασιασμός    | `*`      |
| Διαίρεση           | `/`      |
| Υπόλοιπο Διαίρεσης | `%`      |

### Παραδείγματα:

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << 5 + 2 << endl; // 7
    cout << 5 - 2 << endl; // 3
    cout << 5 * 2 << endl; // 10
    cout << 5 / 2 << endl; // 2
    cout << 5 % 2 << endl; // 1
    return 0;
}
```

#### Υπόλοιπο Διαίρεσης

Το υπόλοιπο διαίρεσης είναι το υπόλοιπο μιας ευκλείδειας διαίρεσης.
Για παράδειγμα, όταν διαιρούμε το 5 με το 2, το αποτέλεσμα είναι 2 με υπόλοιπο 1.

## Τελεστές σύγκρισης

Στη C++ μπορεί να θέλουμε να συγκρίνουμε 2 αριθμούς. Αυτό δεν είναι ιδιαίτερα χρήσιμο σε αυτό το στάδιο,
αλλά είναι χρήσιμο για την εκτέλεση συνθηκών που θα δούμε σε άλλο μάθημα.

Όταν κάνουμε μια σύγκριση, θα έχουμε αποτέλεσμα 0 ή 1.
Το 0 σημαίνει ότι η σύγκριση δεν ισχύει και 1 σημαίνει ότι η σύγκριση είναι σωστή.

Υπάρχουν 6 τελεστές σύγκρισης στη C++:

| Τελεστής              | Σύμβολο  |
|-----------------------|----------|
| Μικρότερος από        | `<`      |
| Μεγαλύτερος από       | `>`      |
| Ίσος με               | `==`     |
| Μικρότερος ή ίσος με  | `<=`     |
| Μεγαλύτερος ή ίσος με | `>=`     |
| Διάφορος από          | `!=`     |

### Παραδείγματα:

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << (5 < 2) << endl; // 0
    cout << (5 > 2) << endl; // 1
    cout << (5 == 2) << endl; // 0
    cout << (5 <= 2) << endl; // 0
    cout << (5 >= 2) << endl; // 1
    cout << (5 != 2) << endl; // 1
    return 0;
}
```

## Λογικοί τελεστές

Στη C++ μπορούμε να συνδυάσουμε πολλές συγκρίσεις μεταξύ τους χρησιμοποιώντας λογικούς τελεστές.
Ας δούμε τους λογικούς τελεστές και τι κάνει ο καθένας:

| Τελεστής | Σύμβολο | Περιγραφή                                                                    |
|----------|---------|------------------------------------------------------------------------------|
| Και      | `&&`    | Επιστρέφει 1 αν και μόνο αν και οι 2 συνθήκες είναι αληθείς.                 |
| Ή        | `\|\|`  | Επιστρέφει 1 αν και μόνο αν τουλάχιστον μία από τις 2 συνθήκες είναι αληθής. |
| Όχι      | `!`     | Επιστρέφει 1 αν και μόνο αν η συνθήκη είναι ψευδής.                          |

### Παραδείγματα:

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << ((5 < 2) && (5 > 2)) << endl; // 0
    cout << ((5 < 2) || (5 > 2)) << endl; // 1
    cout << !(5 < 2) << endl; // 1
    return 0;
}
```

## Μαθηματικές συναρτήσεις

Στη C++ μπορούμε να χρησιμοποιήσουμε μαθηματικές συναρτήσεις για να κάνουμε πολύπλοκους υπολογισμούς.
Αυτές οι συναρτήσεις απαιτούν τη βιβλιοθήκη `cmath`.

### Προσθήκη βιβλιοθήκης

Για να προσθέσουμε μια βιβλιοθήκη στο πρόγραμμά μας, χρησιμοποιούμε την εντολή `#include`, όπως και με το `iostream`.

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    // Κώδικας
    return 0;
}
```

### Συναρτήσεις

Ας δούμε τις συναρτήσεις που παρέχει η βιβλιοθήκη `cmath`:

| Συνάρτηση | Περιγραφή                                                        |
|-----------|------------------------------------------------------------------|
| `abs()`   | Επιστρέφει την απόλυτη τιμή ενός αριθμού.                        |
| `sqrt()`  | Επιστρέφει την τετραγωνική ρίζα ενός αριθμού.                    |
| `pow()`   | Επιστρέφει την τιμή ενός αριθμού στην δύναμη ενός άλλου αριθμού. |
| `sin()`   | Επιστρέφει τον ημιτόνιο ενός αριθμού.                            |
| `cos()`   | Επιστρέφει τον συνημίτονο ενός αριθμού.                          |
| `tan()`   | Επιστρέφει την εφαπτομένη ενός αριθμού.                          |
| `asin()`  | Επιστρέφει τον αντίστροφο ημιτόνιο ενός αριθμού.                 |
| `acos()`  | Επιστρέφει τον αντίστροφο συνημίτονο ενός αριθμού.               |
| `atan()`  | Επιστρέφει τον αντίστροφο εφαπτομένη ενός αριθμού.               |
| `log()`   | Επιστρέφει τον φυσικό λογάριθμο ενός αριθμού.                    |

#### Παραδείγματα:

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    cout << abs(-5) << endl; // 5
    cout << sqrt(25) << endl; // 5
    cout << pow(5, 2) << endl; // 25
    cout << sin(0) << endl; // 0
    cout << cos(0) << endl; // 1
    cout << tan(0) << endl; // 0
    cout << asin(0) << endl; // 0
    cout << acos(1) << endl; // 0
    cout << atan(0) << endl; // 0
    cout << log(1) << endl; // 0
    return 0;
}
```
