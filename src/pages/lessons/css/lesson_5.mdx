# Μάθημα 5ο: Padding, Margins και Ύψος/Πλάτος

## Padding

Η CSS έχει 4 κατηγορίες padding:

* `padding-top`
* `padding-right`
* `padding-bottom`
* `padding-left`

Το padding χρησιμοποιείται για να προσθέσετε έναν χώρο γύρω από το περιεχόμενο ενός στοιχείου. 

Η τιμή ενός padding μπορεί να είναι στις εξής μορφές:

* Μήκος - Π.χ. `10px`
* Ποσοστό - Π.χ. `10%`

### Παράδειγμα

```
div {
  padding-top: 50px;
  padding-right: 30px;
  padding-bottom: 50px;
  padding-left: 80px;
}
```

## Margins

Η CSS έχει 4 κατηγορίες margin, όπως και το padding:

* `margin-top`
* `margin-right`
* `margin-bottom`
* `margin-left`

Το margin χρησιμοποιείται για να προσθέσετε έναν χώρο γύρω από το περιεχόμενο ενός στοιχείου.

Η τιμή ενός margin μπορεί να είναι στις εξής μορφές:

* Μήκος - Π.χ. `10px`
* Ποσοστό - Π.χ. `10%`

### Παράδειγμα

```
div {
  margin-top: 50px;
  margin-right: 30px;
  margin-bottom: 50px;
  margin-left: 80px;
}
```

## Ύψος και Πλάτος

Η CSS έχει 2 κατηγορίες για το ύψος και το πλάτος:

* `height`
* `width`

Με το ύψος και το πάτος μπορούμε να ορίσουμε το ύψος και το πλάτος ενός στοιχείου.

Η τιμή ενός ύψους ή πλάτους μπορεί να είναι στις εξής μορφές:

* Μήκος - Π.χ. `10px`
* Ποσοστό - Π.χ. `10%`
* `auto` - Αυτόματο ύψος ή πλάτος

### Παράδειγμα

```
div {
  height: 200px;
  width: 50%;
}
```
