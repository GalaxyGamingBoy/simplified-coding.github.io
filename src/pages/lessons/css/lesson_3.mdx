# Μάθημα 3: Υπόβαθρο

Η CSS έχει 5 βασικές ιδιότητες με υπόβαθρο:
* `background-color`: Χρησιμοποιείται για να ορίσετε το χρώμα του υποβάθρου.
* `background-image`: Χρησιμοποιείται για να ορίσετε μια εικόνα ως υπόβαθρο.
* `background-repeat`: Χρησιμοποιείται για να ορίσετε αν η εικόνα υποβάθρου θα επαναλαμβάνεται ή όχι.
* `background-position`: Χρησιμοποιείται για να ορίσετε τη θέση της εικόνας υποβάθρου.
* `background-attachment`: Χρησιμοποιείται για να ορίσετε αν η εικόνα υποβάθρου θα παραμένει στην ίδια θέση ή θα κυλάει μαζί με τη σελίδα.

## Χρώμα υποβάθρου

Το έχουμε αναφέρει ήδη στο μάθημα 2ο, αλλά ας κάνουμε μια επανάληψη.

Στην CSS μπορούμε να ορίσουμε ένα χρώμα ως υπόβαθρο για έναν τίτλο, παράγραφο και άλλα.

Για παράδειγμα, αν θέλουμε να ορίσουμε το χρώμα υποβάθρου του τίτλου σε κόκκινο, θα γράψουμε:

### Παράδειγμα

```
h1 {
  background-color: red;
}
```

## Εικόνα υποβάθρου

Μπορούμε επίσης να ορίσουμε μια εικόνα ως υπόβαθρο.

### Παράδειγμα

```
body {
    background-image: url("quack.mp4");
}
```

## Επανάληψη εικόνας υποβάθρου

Αν η εικόνα υποβάθρου είναι μικρότερη από το στοιχείο HTML που την περιέχει, η εικόνα θα επαναλαμβάνεται μέχρι να καλύψει το στοιχείο.

Μπορούμε να αποτρέψουμε την επανάληψη της εικόνας υποβάθρου με την ιδιότητα `background-repeat`.

### Παράδειγμα

```
body {
    background-image: url("quack.mp4");
    background-repeat: repeat-x;
}
```

Παρατηρούμε πως ορίζουμε η εικόνα να επαναλαμβάνεται μόνο οριζόντια με την τιμή `repeat-x`.
Θα μπορούσαμε να την επαναλάβουμε κάθετα με την τιμή `repeat-y`.

## Θέση εικόνας υποβάθρου

Μπορούμε επίσης να ορίσουμε τη θέση της εικόνας υποβάθρου με την ιδιότητα `background-position`.

### Παράδειγμα

```
body {
    background-image: url("quack.mp4");
    background-repeat: repeat-x;
    background-position: right top;
}
```
