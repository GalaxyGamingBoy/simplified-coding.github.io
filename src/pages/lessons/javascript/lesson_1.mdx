# Μάθημα 1ο: Βασική δομή της JS

## Σχόλια

Τα σχόλια στην JS αρχίζουν με `// ΣΧΟΛΙΑ` 

### Παράδειγμα

```js
// ΣΧΟΛΙΟ
```

```js
/* 
ΣΧΟΛΙΟ
ΣΧΟΛΙΟ
ΣΧΟΛΙΟ
*/
```

## `<script>`

Το `<script></script>` βρίσκετε στό τέλος κάθε `<body></body>`. Μέσα στο `<script></script>` μπορούμε να γράψουμε JavaScript

### Παράδειγμα

```html
<!DOCTYPE html>
<html>
    <head>
        <title>JS Sample</title>
    </head>
    <body>
        <h1>Hello World!</h1>
        <script>
            // JS ΕΔΏ
        </script>
    </body>
</html>
```

## Console.log
Με το console.log μπορούμε να γράψουμε στο Web Console.

### Παράδειγμα

```html
<script>
    console.log("Hello World")
</script>
```
