## Εγκατάσταση του Arduino IDE

Πριν αρχίσουμε να προγραμματίζουμε το Arduino, 
πρέπει να εγκαταστήσουμε το Arduino IDE. 
Το Arduino IDE είναι ένα πρόγραμμα που μας επιτρέπει να γράφουμε και να ανεβάζουμε κώδικα στο Arduino. 

### Windows

1. Πατήστε στον σύνδεσμο για να αρχίσει η εγκατάσταση ![εδώ](https://downloads.arduino.cc/arduino-1.8.19-windows.exe)
2. Όταν η εγκατάσταση τελειώσει, ανοίξτε το αρχείο Arduino IDE
3. Εαν ζητηθούν δικαιώματα διαχειριστή, πατήστε Ναι
4. Αρχικά ανοίγει ένα παράθυρο με την άδεια χρήσης του Arduino IDE. Πατήστε "Ι Agree".
5. Στη συνέχεια, πατήστε "Next" στο παράθυρο που ακολουθεί.
6. Στο επόμενο παράθυρο, πατήστε "Install". 
7. Το Arduino IDE θα κατεβάσει μερικούς drivers. Κάποιες εκδόσεις των Windows θα σας ζητήσουν να επιβεβαιώσετε την εγκατάσταση των drivers. Πατήστε "Install" ή "Ναι" για να συνεχίσετε.
8. Το πρόγραμμα έχει εγκατασταθεί. Μπορείτε τώρα να το ανοίξετε από την επιφάνεια εργασίας ή από το μενού "Έναρξη".

### Mac

1. Πατήστε στον σύνδεσμο για να αρχίσει η εγκατάσταση ![εδώ](https://downloads.arduino.cc/arduino-1.8.19-macosx.zip)
2. Βρίσκουμε το συμπιεσμένο zip αρχείο που κατεβάσαμε και το αποσυμπιέζουμε σε µία θέση της επιλογής µας
3. Αντιγράψτε το Arduino application στο φάκελο Downloads ή άλλη επιλεγμένη θέση. 
4. Στο σημείο αυτό το πρόγραμμα Arduino μπορεί να εκτελεστεί κανονικά.

### Linux

για να κατεβάσουν το πρόγραμμα μόνοι τους!

## Φόρτωση προγράμματος στο Arduino

Για να φορτώσουμε ένα πρόγραμμα στο Arduino,
πρέπει πρώτα να επιλέξουμε το σωστό μοντέλο του Arduino.
Σε αυτά τα μαθήματα, θα χρησιμοποιήσουμε το Arduino Uno.

Για να επιλέξουμε το μοντέλο του Arduino, 
πρέπει να ανοίξουμε το Arduino IDE και να πάμε στο μενού "Εργαλεία" 
και στη συνέχεια στην επιλογή "Πλακέτα".
Από εκεί, επιλέγουμε το μοντέλο του Arduino που θέλουμε να χρησιμοποιήσουμε.

Πρέπει επιπλέον, να επιλέξουμε τη σωστή θύρα στην οποία είναι συνδεδεμένο το Arduino.
Για να το κάνουμε αυτό, πάμε στο μενού "Εργαλεία" και στη συνέχεια στην επιλογή "Θύρα".
Από εκεί, επιλέγουμε τη θύρα στην οποία είναι συνδεδεμένο το Arduino.

Τέλος, για να ανεβάσουμε ένα πρόγραμμα,
πρέπει να πατήσουμε το κουμπί "Ανεβάστε" στο Arduino IDE.
